Push v1.0

Welcome to the push messaging example from Jason. The button below needs to clicked to subscribe to push notifications.

Once you've subscribed we store the long piece of code below, next to your name in the database. You can choose to subscribe on as many devices as you choose. Each new device would add a new piece of code next to your name in the database. It looks like a url, right? We can send messages to that URL and it will arrive on your device, regardless of the browser being open or not.
